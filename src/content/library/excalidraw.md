## Overview
Excalidraw is a whiteboard tool that lets you easily sketch diagrams that have a hand-drawn feel. It's built on top of the browser and focuses on simplicity and security.

## Features
- **Aesthetic:** Beautiful, hand-drawn look for technical sketches.
- **Privacy:** End-to-end encrypted collaboration.
- **Versatile:** Export to multiple high-quality formats.

## Use Cases
- Designing system architectures.
- Visual brainstorming in remote teams.
- Creating casual blog illustrations.
