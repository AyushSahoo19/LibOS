## Overview
Ruflo is an advanced orchestration layer for Claude Code. It allows a single human operator to spawn and coordinate a fleet of specialized subagents—Architects, Coders, Testers—that work together within the Claude Code CLI.

## Features
- **Dynamic Role Assignment:** Assign specific personas and constraint sets to subagents.
- **Inter-Agent Communication:** Allows subagents to pass context and deliverables between themselves.
- **Context Consolidation:** Automatically summarizes agent logs to stay within token limits.

## Use Cases
- Complex engineering problems requiring multi-disciplinary expertise.
- Automated end-to-end software delivery pipelines.
- Researching and mapping large unknown codebases.
