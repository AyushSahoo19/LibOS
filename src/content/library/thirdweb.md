## Overview
thirdweb provides a suite of tools for developers to build decentralized applications without needing to write complex smart contract code. It includes SDKs for various languages and a dashboard for managing contracts.

## Features
- **Pre-built Contracts:** Audited contracts for NFTs, Tokens, and Marketplaces.
- **Unified SDK:** A single SDK for Frontend, Backend, and Gaming (Unity/Unreal).
- **Dashboard:** Visual interface to deploy and manage blockchain projects.

## Use Cases
- Rapidly launching NFT collections and marketplaces.
- Onboarding non-crypto users via social login and fiat rails.
- Integrating blockchain features into existing 2D/3D games.
