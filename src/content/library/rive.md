## Overview
Rive is a real-time interactive design and animation tool. The Rive runtime allows you to take your animations and use them in apps, games, and websites with full control over states and interactions.

## Features
- **State Machine:** Design logic directly in the design tool.
- **Tiny Payloads:** Proprietary `.riv` format is extremely small.
- **Wasm Powered:** High performance web runtime using WebAssembly.

## Use Cases
- Interactive game characters for the web.
- State-driven UI components.
- Platform-agnostic design systems.
