## Overview
A high-performance library designed specifically for complex software engineering. Features a modular architecture where each agent is token-optimized for a single vertical (e.g., testing, refactoring, security).

## Features
- **Token Efficiency:** Minimalist prompt design to save on context costs.
- **75+ Focused Plugins:** Direct integrations with modern build tools and CI/CD pipelines.
- **Role-Based Orchestration:** Defines clear boundaries for subagent delegation.

## Use Cases
- Managing large-scale codebase refactors.
- Automated security auditing via specialized agent roles.
- Building custom CI/CD sidecars using Claude Code.
