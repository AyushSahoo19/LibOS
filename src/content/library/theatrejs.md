## Overview
Theatre.js is an animation library with a GUI. It allows you to animate any JS object, whether it's Three.js, React, or just plain HTML/CSS, using a professional-looking timeline and stage.

## Features
- **Visual Editor:** A browser-based editor to keyframe properties in real-time.
- **Extensible:** Orchestrate complex sequences across multiple frameworks.
- **Production Ready:** Small runtime bundle for the final build.

## Use Cases
- Cinematic sequences on web pages.
- Complex data visualization transitions.
- High-end interactive landing pages.
