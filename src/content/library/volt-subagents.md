## Overview
A curated collection of specialized subagent definitions for Claude Code. These high-end "personas" are designed to be spawned from within your main session to handle granular, mission-critical tasks.

## Features
- **The Debugger:** A subagent optimized for stack trace analysis and root-cause identification.
- **The Tester:** Automatically generates Playwright, Vitest, or Pytest suites for new code.
- **The DocGen:** Curates and updates `CLAUDE.md` and `README.md` files based on repository changes.

## Use Cases
- Scaling complex development tasks across multiple specialized roles.
- Improving code quality through automated subagent review stages.
- Maintaining project documentation in real-time.
