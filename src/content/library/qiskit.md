## Overview
Qiskit is an open-source software development kit (SDK) created by IBM for working with quantum computers. It provides tools for creating and manipulating quantum programs and running them on prototype quantum devices on IBM Quantum Experience or on simulators on a local computer.

## Features
- **Circuit Construction:** Rich API to compose complex quantum circuits dynamically.
- **Transpiler:** State-of-the-art quantum circuit transpiler that optimizes circuits for specific hardware topologies.
- **Simulators:** Powerful local simulators (Aer) mimicking realistic noise and decoherence.

## Use Cases
- Quantum machine learning.
- Shor's algorithm and cryptography research.
- Simulating molecular quantum dynamics.
