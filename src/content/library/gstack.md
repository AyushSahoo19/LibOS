## Overview
G-Stack is an opinionated, highly specialized setup for Anthropic's Claude Code, popularized by Garry Tan. It provides a structured set of prompts, tools, and rules that turn a standard LLM coding assistant into a multi-role engineering department.

## Features
- **Role-Playing Personas:** Tools specifically designed to make Claude act as a CEO, Engineering Manager, QA, or Designer depending on the task at hand.
- **Workflow Standardization:** Forces the AI to follow strict engineering best practices, from PR reviews to documentation generation.
- **Out-of-the-box Ready:** Designed to be dropped into any existing repository to immediately upgrade the capabilities of Claude Code.

## Use Cases
- Solo founders who need specialized "virtual" team members.
- Enforcing strict coding and documentation standards automatically.
- High-level "Vibe Coding" where the developer acts as the conductor rather than the typist.
