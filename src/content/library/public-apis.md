## Overview
A collective list of free APIs for use in software and web development. This is a massive directory covering nearly every category imaginable, from weather and finance to cats and anime.

## Features
- **Massive Directory:** Thousands of verified free-to-use APIs.
- **Categorized:** Search by Business, Music, Games, News, Social, etc.
- **Support Details:** Indicates if an API supports HTTPS, Auth, and CORS.

## Use Cases
- Finding data sources for portfolio projects.
- Rapid prototyping with real-world data.
- Learning how to integrate diverse 3rd party services.
