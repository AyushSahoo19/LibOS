## Overview
Originally a personal to-do list for a developer who wanted to transition from Web Dev to Software Engineer at Amazon. It has evolved into a massive community resource for preparing for technical interviews at top-tier tech companies.

## Features
- **Comprehensive:** Covers DS, Algorithms, System Design, Operating Systems, etc.
- **Daily Plan:** Offers a structured path for monthly or yearly preparation.
- **Resource Curated:** Best videos and blogs for every single topic.

## Use Cases
- Preparing for interviews at Google, Meta, Amazon, or Netflix.
- Transitioning into platform or systems engineering.
- Rapidly catching up on specialized CS topics.
