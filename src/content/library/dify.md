## Overview
Dify is an open-source LLM app development platform. Its intuitive interface combines AI workflow, RAG pipeline, agent capabilities, model management, and observability features, allowing you to quickly go from prototype to production.

## Features
- **Visual Workflow:** Design complex agentic loops using a drag-and-drop editor.
- **RAG Engine:** Built-in support for cleaning and indexing documentation for use by agents.
- **Model Agnostic:** Support for OpenAI, Anthropic, Llama 3, and more.

## Use Cases
- Building enterprise-grade AI assistants.
- Orchestrating complex data pipelines involving multiple LLM steps.
- Creating and managing a library of custom agent tools.
