## Overview
Hoppscotch is an open-source lightweight API development suite. It was built from the ground up with ease of use and accessibility in mind.

## Features
- **Fast UI:** Reactive web-based interface.
- **Protocol Support:** HTTP, WebSocket, GraphQL, and gRPC.
- **Team Friendly:** Open-source collaboration features.

## Use Cases
- Testing and documenting web APIs.
- Collaborative API development.
- Debugging real-time networking streams.
