## Overview
The definitive curated index for Claude Code — containing every verified skill, hook, slash-command, subagent, memory plugin, and orchestration pattern in the ecosystem. If something is high-quality and works with Claude Code, it's listed here.

## Features
- **Skills Directory:** Every drop-in skill folder for `mnt/skills/` across languages and domains.
- **Slash Commands:** Community-built command patterns for interrogation, scaffolding, testing, PR review, and more.
- **Orchestrators:** Multi-agent coordination patterns, subagent delegation, memory consolidation.
- **Hooks:** Pre/post tool hooks for safety gates, logging, and custom tool routing.

- Finding reference patterns for building your own skills and slash commands.
- Understanding the Claude Code ecosystem before starting a new project.
